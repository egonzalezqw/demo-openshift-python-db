# Demo OpenShift Python + PostgreSQL

Aplicación Flask conectada a base de datos PostgreSQL.

## Features
- UI web
- Insertar mensajes
- Ver mensajes
- API JSON

## Endpoints
- / → UI
- /add → guardar mensaje
- /api/messages → JSON
- /health → status

## Variables OpenShift
DB_HOST
DB_NAME
DB_USER
DB_PASS
